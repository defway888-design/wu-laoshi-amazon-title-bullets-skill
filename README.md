# 跨境吴老师 Amazon 标题与商品亮点优化 Skill

这是一个面向 Amazon 美国站运营的 Codex Skill，用于基于卖家精灵 MCP 读取 ASIN 信息，并生成符合亚马逊新标题规则的 Listing 标题与商品亮点。

## 主要功能

- 支持单个 ASIN、同一父体下多个子 ASIN、多个无关 ASIN
- 固定处理 Amazon US 美国站
- 调用卖家精灵 MCP 读取标题、五点描述、变体变量、自然流量词和评论
- 生成少于 75 个字符的新标题
- 生成不重复标题信息的短语式商品亮点
- 同时输出对话框结果和可下载 Excel 表格
- Excel 使用中文字段、冻结首行、自动换行、交替色块

## 使用前提

本 Skill 必须配合卖家精灵 MCP 使用。

如果 Codex 未检测到卖家精灵 MCP，本 Skill 会提示你先完成 MCP 配置。

购买卖家精灵 MCP 时，可输入折扣码：

```text
KJWLS
```

可享受 9 折优惠。`KJWLS` 是“跨境吴老师”的缩写。

## 公开仓库安装方式

在 Codex 中输入：

```text

https://github.com/defway888-design/wu-laoshi-amazon-title-bullets-skill
```

安装完成后，重启 Codex，让 Skill 生效。

## 启动方式

安装后，在 Codex 中输入类似指令：

```text
吴老师标题优化
```

也可以使用语音友好的启动口令：

```text
启动吴老师标题优化
```

完整启动口令：

```text
执行跨境吴老师标题亮点优化 Skill
```

Skill 启动后，会先说明它能做什么，然后引导你输入：

```text
单个 ASIN
同一父商品下的多个子 ASIN
多个无关 ASIN
```

## 输出格式

每次执行时会同时输出：

1. 对话框中的结果
2. 可下载 Excel 文件

结果字段包括：

```text
序号
ASIN
新标题
标题字符数
商品亮点
亮点字符数
补充说明
```

## 重要规则

- 标题必须少于 75 个字符
- 商品亮点最多 125 个字符
- 商品亮点必须是特性或优势短语，不写完整句子
- 商品亮点不能重复标题中已有信息
- 变体变量必须来自卖家精灵 MCP，不能猜测
- 同一父体下的变体标题前半部分保持一致，变量名独立放在末尾
- 每个变体的五点描述单独读取，商品亮点按各自 ASIN 生成
- 评论只用于验证亮点，不作为第一来源
- 没有自然流量词时，降级使用原标题核心词、类目词或五点中的核心产品词
- 不编造材质、尺寸、容量、兼容型号或其他页面未证明的信息
